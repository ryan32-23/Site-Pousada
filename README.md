# Pousada Secreta - Site Institucional

Site estático de uma pousada fictícia em Angra dos Reis, criado com HTML e CSS puros.

## 🧭 Visão Geral
Este projeto é um site institucional simples com:
- Página inicial (`index.html`) com informações sobre a pousada.
- Página de quartos (`quartos.html`) com detalhes de tipos de acomodações.
- Navegação no topo para seções *Sobre*, *Rota* e *Quartos*.
- Mapa embutido do Google Maps.
- Seção de contato com endereço, telefone e link para Booking.

## 📁 Estrutura de arquivos
- `index.html` - página inicial do site
- `quartos.html` - página com detalhes dos quartos
- `css/estilo.css` - estilo principal do site
- `css/estilo-quartos.css` - estilo da página de quartos
- `assets/` - imagens e ícones usados no site

## 🚀 Como usar
1. Abra `index.html` no navegador (duplo clique ou arrasta no browser).
2. Navegue entre as seções usando o menu no topo.
3. Para editar:
   - Modifique textos no `index.html` e `quartos.html`.
   - Ajuste layout e cores nos arquivos CSS.

## ⚙️ Funcionalidades incluídas
- Layout responsivo (adaptável para desktop e mobile).
- Cards de quartos com imagem + link de navegação.
- Seções de atração do site: *Bem-vindo*, *Rota*, *Sobre*.
- Barra fixa no topo com ancoragem para seções.
- Rodapé com contatos e chamadas de ação.
